# ML Company Workflow - Customer Churn Prediction

## Overview

Production-grade machine learning pipeline for predicting customer churn. This repository implements a complete MLOps workflow with automated training, evaluation, deployment, and monitoring using GitHub Actions.

## Features

- **Automated ML Pipeline**: End-to-end automation from data ingestion to model deployment
- **Multi-model Training**: Parallel training of Logistic Regression, Random Forest, and XGBoost
- **Model Evaluation**: Comprehensive metrics (Accuracy, Precision, Recall, F1-Score)
- **Model Registry**: Automatic selection and versioning of best models
- **FastAPI Deployment**: REST API for real-time predictions
- **Data Drift Detection**: Continuous monitoring of data distribution changes
- **Automated Retraining**: Scheduled and trigger-based model retraining
- **Docker Containerization**: Portable and scalable deployment
- **Comprehensive Testing**: Unit tests, integration tests, and API tests
